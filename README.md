Kavach AI

Kavach AI is an AI-powered scam and fraud detection project created during Global Hack Week: Hacking for Good.

Problem

Online scams and fraudulent messages are becoming increasingly common and many users struggle to identify suspicious content.

Solution

Kavach AI helps users analyze messages and identify potential scams by providing a risk score and explaining suspicious patterns.

Features

- Scam message detection
- Risk score analysis
- Explainable AI output
- Simple and accessible interface
- Fast message checking

Technologies Used

- HTML
- CSS
- JavaScript
- Python
- Gemini API
